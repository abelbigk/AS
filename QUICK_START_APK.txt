================================================================================
                    IF YOU GET ERRORS IN ANDROID STUDIO
================================================================================

MOST COMMON: jlink.exe failure

FIX 1 (Do this first):
─────────────────────
1. File → Settings
2. Build, Execution, Deployment → Build Tools → Gradle
3. Change Gradle JDK to: JDK 17
4. Click Apply & OK
5. File → Invalidate Caches → Restart
6. Try building again

FIX 2 (If Fix 1 doesn't work):
──────────────────────────────
1. Build → Clean Project
2. Close Android Studio
3. Delete: C:\Users\abelb\.gradle\caches\transforms-3
4. Reopen Android Studio
5. Try building again

FIX 3 (Last resort):
────────────────────
1. Tools → SDK Manager
2. Uncheck "Android 14.0 (API 34)"
3. Apply (uninstall)
4. Recheck "Android 14.0 (API 34)"
5. Apply (reinstall)
6. Try building again
