🔧 IF SOMETHING GOES WRONG
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Problem: "Could not sync gradle"
  Solution: 
    1. File → Invalidate Caches
    2. Select "Invalidate and Restart"
    3. Wait for restart (2-3 min)
    4. Try sync again

  Problem: "Build failed"
  Solution:
    1. Build → Clean Project
    2. Wait 30 seconds
    3. Build → Build APK(s)
    4. Try again

  Problem: "compileSdkVersion not specified"
  Solution:
    1. File → Project Structure
    2. Set Compile SDK Version to 34
    3. Apply, close, and sync again
