================================================================================
                      TROUBLESHOOTING
================================================================================

If Gradle Sync is Slow:
  1. File → Invalidate Caches
  2. Click "Invalidate and Restart"
  3. Wait for sync to complete

If Build Fails:
  1. Build → Clean Project
  2. Build → Rebuild Project
  3. Run app again

If Emulator Not Starting:
  1. Open Android Studio's AVD Manager
  2. Select your emulator
  3. Click "Play" button
  4. Wait for emulator to boot
  5. Run app

If App Crashes:
  1. Check Logcat (bottom panel)
  2. Look for error messages
  3. Make sure backend (https://as-wryo.onrender.com) is running
  4. Restart app
