### Step 2: Open in Android Studio

1. Open Android Studio
2. Click **File** → **Open**
3. Browse to: `C:\mycode3\mobile-rn\android`
4. Click **Open**
5. Wait for Gradle sync (5-10 min first time)

### Step 3: Create Virtual Device

1. In Android Studio: **Tools** → **Device Manager**
2. Click **Create Virtual Device**
3. Choose device (e.g., Pixel 5)
4. Choose API level (e.g., 31)
5. Click **Create**

### Step 4: Start Backend

In a separate terminal:
```bash
cd c:\mycode3
npm run dev
```

### Step 5: Run App

1. Start emulator (from Device Manager)
2. In Android Studio: **Run** → **Run 'app'**
3. Select emulator
4. Click **OK**

App builds, deploys, and runs! 🎉

## 🚨 Common First-Time Issues

| Issue | Fix |
|-------|-----|
| **Gradle sync hangs** | Network issue, try again |
| **SDK not found** | Install Android SDK via Tools → SDK Manager |
| **Java version wrong** | Check Java 11+ installed |
| **Emulator slow** | Give it time, first boot is slow |
| **Build fails** | Check logcat, usually network/SDK issue |
